# 西红柿炒鸡蛋 - 经典家常菜

调用 '.claude/skills/tomato-and-egg/scripts/helloWorld.js' 来测试技能是否加载成功。

## 用料

| 食材 | 用量 |
|------|------|
| 鸡蛋 | 3个 |
| 西红柿 | 2个（约300g） |
| 食用油 | 适量 |
| 盐 | 适量 |
| 白糖 | 少许（提鲜） |
| 葱花 | 少许 |

## 步骤

### 第一步：准备食材
1. 鸡蛋打入碗中，加入少许盐，搅拌均匀
2. 西红柿洗净，切成小块备用
3. 葱切成葱花

### 第二步：炒鸡蛋
1. 热锅凉油，油量稍多一些
2. 倒入蛋液，快速翻炒至凝固
3. 鸡蛋炒至金黄色嫩滑状态后盛出备用

### 第三步：炒西红柿
1. 锅中留底油，放入葱花爆香
2. 倒入西红柿块，中火翻炒
3. 炒至西红柿出汁变软
4. 加入少许白糖提鲜
5. 根据口味加入适量盐

### 第四步：合炒
1. 倒入炒好的鸡蛋
2. 快速翻炒均匀
3. 让鸡蛋充分吸收西红柿的汤汁
4. 翻炒均匀后即可出锅

## 小贴士

- 🥚 鸡蛋要炒得嫩一些，不要炒太老
- 🍅 西红柿要炒出汁水才好吃
- 🍬 加少许糖可以中和西红柿的酸味
- 🔥 全程大火快炒，保持食材鲜嫩
- ⏰ 整个过程约5-8分钟

## 成品特点

- 颜色：红黄相间，色泽诱人
- 口感：鸡蛋嫩滑，西红柿酸甜
- 营养：富含蛋白质和维生素
